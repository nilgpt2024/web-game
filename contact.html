GameHub 是一个收录了 54 款精品 HTML5 游戏的开源合集项目。所有游戏完全免费，无需下载安装，直接在浏览器中即可畅玩。只需访问主页 (index.html)，浏览游戏分类，点击感兴趣的游戏卡片即可开始游玩。

我们非常欢迎社区贡献！您可以：

  1. Fork 本项目到您的 GitHub 账户
  2. 按照现有游戏目录结构开发新游戏
  3. 确保游戏使用 HTML5 + CSS3 + JavaScript 纯前端技术
  4. 编写 README.md 说明文档
  5. 提交 Pull Request 进行代码审查

详细的贡献指南请查看 CONTRIBUTING.md 文件。

如果您在使用过程中发现任何 Bug 或有改进建议，推荐通过以下方式反馈：

  • GitHub Issues（推荐）：访问 Issues 页面，使用提供的模板提交 Bug Report 或 Feature Request
  • 本页表单：填写上方的反馈表单，我们会整理后跟进处理
  • Email/QQ：对于紧急问题可直接通过 Email 或 QQ 联系

请在反馈中尽可能详细地描述问题复现步骤、预期行为和实际行为，这将帮助我们更快定位和解决问题。

GameHub 采用现代 Web 技术构建，具有良好的跨平台兼容性：

  • 桌面端：Chrome、Firefox、Safari、Edge 等主流浏览器的最新两个版本
  • 移动端：iOS Safari、Android Chrome 等移动浏览器
  • 响应式设计：自适应不同屏幕尺寸，手机、平板、电脑均可流畅体验

部分游戏可能依赖特定 API（如 Canvas、Web Audio），建议使用较新版本的现代浏览器以获得最佳体验。

GameHub 项目采用 MIT 开源协议，允许自由使用、修改和分发。但请注意：

  • 项目中部分游戏可能引用了第三方资源或遵循其他许可协议
  • 商业使用前请仔细阅读各游戏的 README 和 LICENSE 信息
  • 如需商业授权或有特殊需求，欢迎通过 Email 或 QQ 与我们洽谈

我们鼓励开源共享精神，也尊重每一位贡献者的劳动成果。
