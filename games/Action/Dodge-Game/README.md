# Dodge Game - 躲避障碍

控制角色左右移动，躲避从天而降的障碍物，坚持更长时间获得更高分数！

## 🎮 操作方式

| 平台 | 操作 |
|------|------|
| 键盘 | `←` `→` 方向键 或 `A` `D` 键左右移动 |
| 触摸 | 触摸/拖动屏幕左右两侧控制移动 |
| 开始/重启 | 点击「开始游戏」按钮或按 `空格` / `回车` |

## ✨ 游戏特色

- **3D 透视视角** - 障碍物从远处飞来逐渐变大，营造纵深感
- **多种障碍物类型** - 岩石、陨石、水晶、炸弹，各有不同速度和大小
- **粒子爆炸特效** - 碰撞时产生华丽的粒子爆炸效果
- **难度递增系统** - 每获得100分升一级，速度加快、障碍物增多
- **玩家发光特效** - 角色带有动态光晕和运动轨迹残影
- **实时计分 + 最高分记录** - 使用 localStorage 持久化存储
- **响应式设计** - 支持桌面端和移动端

## 🔧 技术实现

- **Canvas 2D 渲染** - 高性能游戏画面绘制
- **3D 透视模拟** - 通过缩放比例和透明度变化模拟深度
- **网格透视线** - 增强空间感的辅助视觉元素
- **星空背景** - 动态闪烁的星星点缀
- **粒子系统** - 支持圆形粒子和文字粒子
- **碰撞检测** - AABB 缩放边界框检测
- **平滑移动** - 线性插值 (lerp) 实现流畅的角色跟随
