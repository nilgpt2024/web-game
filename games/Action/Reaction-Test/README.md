# ⚡ Reaction Test - 反应测试

测试你的极限反应速度！支持三种游戏模式，精确到毫秒级的反应时间测量。

## 🎮 游戏模式

### 🎯 经典模式 (Classic)
最纯粹的反应速度测试。点击开始后等待屏幕变绿，以最快速度点击屏幕。

**规则：**
- 点击或按空格键开始测试
- 屏幕显示红色时 **请勿点击**（太早点击算失败）
- 屏幕变为绿色后尽快点击
- 系统精确记录你的反应时间（毫秒级）

### 🎨 颜色模式 (Color)
在经典模式基础上增加颜色识别维度，考验视觉处理+反应的综合能力。

**规则：**
- 开始前会显示一个目标颜色
- 等待屏幕变为目标颜色后尽快点击
- 需要同时识别颜色和判断时机

### 🔘 序列模式 (Sequence)
类似 Simon Says 的序列记忆+反应测试，按钮会依次亮起，你需要按相同顺序点击。

**规则：**
- 观察亮起的按钮序列
- 序列播放完后，在限定时间内按顺序重复
- 每回合序列长度 +1
- 完成 5 回合后计算平均反应时间

## 🛠️ 技术特性

- **CSS 3D 按钮**：立体按下效果、发光边框、悬停动画
- **脉冲动画**：等待状态和就绪状态的动态反馈
- **渐变背景**：深色科技风主题
- **localStorage 存储**：历史记录、最佳成绩自动保存
- **响应式设计**：适配桌面端、平板和手机
- **国际化支持**：中英文切换
- **键盘支持**：空格键快速操作

## 🎯 操作方式

- **鼠标/触屏**：点击反应区域或序列按钮
- **键盘**：空格键触发反应（经典/颜色模式）
