# Space Shooter - 太空射击

驾驶你的战斗机穿越浩瀚星海，击败外星舰队和终极Boss！

## 游戏特色

- **4种敌人类型**：侦察机、战斗机、重装舰、精英单位，各有不同血量、速度和攻击模式
- **Boss战系统**：每5波遭遇一次Boss，拥有多种攻击模式（弹幕、追踪、散射等）
- **道具系统**：火力增强、护盾保护、生命恢复、全屏清敌炸弹
- **波次系统**：难度递增，敌人越来越多越来越强
- **粒子特效**：爆炸效果、引擎尾焰、子弹轨迹
- **星空滚动背景**：沉浸式太空体验
- **音效系统**：射击、爆炸、拾取道具等音效
- **响应式设计**：支持桌面端和移动端
- **国际化支持**：中英文切换

## 操作方式

| 按键 | 功能 |
|------|------|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| 空格 | 发射子弹 |
| P | 暂停/继续游戏 |

## 敌人类型

| 类型 | 外观 | 血量 | 速度 | 分数 | 特点 |
|------|------|------|------|------|------|
| 侦察机 Scout | 🛸 椭圆形 | 1 | 快 | 100 | 基础敌人 |
| 战斗机 Fighter | 🔺 三角形 | 3 | 中 | 200 | 标准敌人 |
| 重装舰 Heavy | ⬢ 方形 | 6 | 慢 | 350 | 高血量 |
| 精英单位 Elite | 💀 菱形 | 10 | 快 | 500 | 追踪射击 |

## Boss攻击模式

1. **扇形弹幕** - 同时发射5发扩散子弹
2. **环形弹幕** - 360度发射圆形子弹
3. **双管齐射** - 两门主炮直射
4. **瞄准追踪** - 锁定玩家位置发射

## 道具系统

| 道具 | 颜色 | 效果 |
|------|------|------|
| 🔥 FIRE (火力) | 橙色 | 提升火力等级（最高LV.4），增加子弹数量 |
| 🛡️ SHIELD (护盾) | 蓝色 | 获得护盾保护，抵挡一次伤害 |
| ❤️ HEAL (恢复) | 绿色 | 恢复35点生命值 |
| 💣 BOMB (炸弹) | 红色 | 清除屏幕上所有敌人和敌方子弹 |

## 游戏提示

1. 收集火力道具提升到LV.4可获得最大火力输出
2. Boss战时优先躲避弹幕，寻找安全位置反击
3. 炸弹在敌人密集时使用效果最佳
4. 护盾可以抵挡任何形式的伤害（包括撞击）
5. 波次越高，敌人属性越强，注意走位
