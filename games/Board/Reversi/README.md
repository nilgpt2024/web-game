# 黑白棋 (Reversi / Othello)

经典的黑白棋对战游戏，支持人机对战。

## 游戏规则

- **目标**：在游戏结束时拥有更多己方颜色的棋子
- **黑棋先行**：玩家执黑棋，AI 执白棋
- **落子规则**：必须将棋子放在能夹住对方棋子的位置（横、纵、斜均可）
- **翻转**：被夹住的对方棋子会翻转成你的颜色
- **跳过回合**：如果没有合法移动，则自动跳过该回合
- **结束条件**：当双方都无法移动时，游戏结束

## 功能特性

- 8x8 标准棋盘
- CSS 3D 棋子效果与翻转动画
- 三档 AI 难度（简单 / 中等 / 困难）
- 合法走位提示（可开关）
- 悔棋功能
- 实时分数显示
- 上一步位置高亮
- 响应式设计，支持移动端
- 国际化支持（中文 / English）

## AI 策略

| 难度 | 策略 |
|------|------|
| 简单 | 随机选择合法位置 |
| 中等 | 位置权重评估（角 > 边 > 中间），2层前瞻 |
| 困难 | 强化位置策略，避免角旁危险位，5层前瞻 |

## 技术实现

- **3D 效果**：CSS `perspective` + `transform-style: preserve-3d`
- **翻转动画**：`rotateY(0deg) → rotateY(180deg)` 关键帧动画
- **落子动画**：从上方弹入的 `translateY` + `scale` 组合动画
- **AI 算法**：基于位置权重表 + 角/边策略 + 对手行动预测的启发式评估
