# Solitaire (纸牌接龙 / Klondike)

经典单人纸牌游戏 - 将所有卡牌按花色从 A 到 K 移至目标堆。

## 游戏规则

### 目标
将 52 张牌全部移动到右上角的 4 个目标堆（Foundation），每个目标堆按同一花色从 A 到 K 依次排列。

### 基本操作

| 操作 | 说明 |
|------|------|
| **发牌** | 点击左上角牌堆（Stock）翻牌 |
| **移动卡牌** | 拖拽卡牌到目标位置，或点击选中后点击目标位置 |
| **快速归位** | 双击卡牌自动移到对应的目标堆 |
| **撤销** | 点击撤销按钮或 `Ctrl+Z` 撤销上一步 |

### 牌堆规则

- **Tableau（7列牌堆）**：必须红黑交替、数字递减放置（如：红Q → 黑J → 红10）
- **Foundation（4个目标堆）**：按同花色从 A 开始递增堆叠
- **空位规则**：只有 K 可以放到空的 Tableau 列

### 计分规则

| 操作 | 分数 |
|------|------|
| 移动到目标堆 | +10 |
| 翻开一张暗牌 | +5 |
| 从目标堆移出 | -15 |
| 重发废牌堆 | -20 |

### 功能按钮

- **新游戏**：重新洗牌开始一局新游戏
- **撤销**：撤销最近一步操作（最多100步）
- **自动完成**：当所有牌已翻开时，一键将可移动的牌自动送入目标堆
- **提示**：高亮显示一个可行的移动方案

## 技术特性

- CSS 3D Transforms 实现逼真的卡片翻转动画
- 支持鼠标拖拽和触屏操作
- 完整的 Klondike 规则验证引擎
- 响应式设计，适配桌面端和移动端
- 国际化支持（中文 / English）
- Plus Jakarta Sans 字体 + 青色主题 (#0d9488)
