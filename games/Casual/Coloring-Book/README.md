# Coloring Book - 涂色本

一个精美的涂色游戏，提供多种图案和丰富的颜色选择，释放你的创意！

## 游戏特性

### 10种预设图案
- **小猫** (Cat) - 可爱的猫咪轮廓
- **小狗** (Dog) - 呆萌的小狗形象
- **蝴蝶** (Butterfly) - 优雅的蝴蝶图案
- **小鱼** (Fish) - 活泼的鱼儿
- **房子** (House) - 温馨的小屋
- **大树** (Tree) - 茂盛的大树
- **太阳** (Sun) - 明亮的太阳
- **花朵** (Flower) - 美丽的花朵
- **星星** (Star) - 闪亮的五角星
- **爱心** (Heart) - 浪漫的心形

### 绘画工具
- **填充工具** - 点击区域自动填充颜色（Flood Fill算法）
- **画笔工具** - 自由绘画，支持调节笔刷大小
- **橡皮擦** - 擦除已绘制的内容

### 调色板功能
- 11种预设颜色（红、橙、黄、绿、青、蓝、紫、粉、灰、黑、白）
- 自定义颜色选择器（Color Picker）
- 当前颜色实时显示

### 操作功能
- **撤销** - 支持最多30步撤销历史
- **清除全部** - 一键清空画布
- **保存图片** - 导出为PNG格式下载

### 视觉效果
- 3D立体画布框架（透视变换、悬浮阴影）
- 渐变背景与玻璃态卡片设计
- 工具按钮发光动画
- 颜色选择器缩放交互效果
- 图标浮动动画

### 技术特点
- Canvas 2D 原生绘图
- Flood Fill 填充算法（带容差）
- 双层Canvas架构（线条层 + 填充层）
- 触摸屏支持（移动端适配）
- 国际化支持（中文/英文切换）
- 响应式布局

## 使用方式

1. 选择顶部图案网格中的一个图案
2. 在调色板中选择想要的颜色（或使用自定义颜色选择器）
3. 使用工具栏切换绘画模式：
   - **填充模式**：点击图案区域自动填色
   - **画笔模式**：自由绘制线条
   - **橡皮擦模式**：擦除内容
4. 使用底部操作按钮进行撤销、清除或保存图片

## 技术栈

- HTML5 Canvas API
- CSS3（渐变、阴影、3D变换、动画）
- Vanilla JavaScript（ES6+）
- Font Awesome 6.4（图标库）

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
