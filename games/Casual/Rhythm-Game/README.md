# 音乐节奏游戏 (Rhythm Game)

一个基于 Web 技术的音乐节奏游戏，支持键盘和触摸操作。

## 🎮 游戏特性

- **4 轨道演奏系统** - 使用 D/F/J/K 键或方向键操作
- **3D 视觉效果** - CSS 3D 音符下落、轨道发光、打击特效
- **精准判定系统** - Perfect / Good / Miss 三级判定
- **连击倍率系统** - 连击越高，分数倍率越大（最高 3 倍）
- **多难度曲目** - 4 首预设曲目（Easy/Normal/Hard/Extreme）
- **音效反馈** - 使用 Web Audio API 生成打击音效
- **响应式设计** - 支持桌面和移动设备
- **国际化支持** - 中英文界面切换
- **统一风格** - Plus Jakarta Sans 字体 + 青色主题 (#0d9488)

## 🎯 操作说明

### 键盘操作
| 轨道 | 主按键 | 备用按键 |
|------|--------|----------|
| 1 (左) | D | ← |
| 2 (左中) | F | → |
| 3 (右中) | J | ↑ |
| 4 (右) | K | ↓ |

### 触摸操作
移动设备自动显示 4 个触摸区域，点击对应区域即可击打音符。

### 其他控制
- `空格` / `ESC` - 暂停/继续游戏

## 🎵 曲目信息

| 难度 | 名称 | BPM | 时长 | 特点 |
|------|------|-----|------|------|
| Easy | 轻松节拍 | 100 | 30s | 适合新手，音符稀疏 |
| Normal | 动感旋律 | 130 | 35s | 标准难度，节奏明快 |
| Hard | 极速狂飙 | 160 | 40s | 高难度，需要快速反应 |
| Extreme | 地狱模式 | 200 | 45s | 极限挑战，密集音符 |

## ⭐ 判定系统

| 判定 | 时间窗口 | 分数 | 说明 |
|------|----------|------|------|
| Perfect | ±50ms | 100 × 倍率 | 完美时机！ |
| Good | ±120ms | 50 × 倍率 | 还不错 |
| Miss | 超时 | 0 | 错过了... |

## 🛠️ 技术实现

### 核心技术
- **HTML5** - 语义化结构
- **CSS3** - 3D transforms、animations、gradients
- **JavaScript ES6+** - Class-based 游戏引擎
- **Web Audio API** - 程序化音效生成
- **RequestAnimationFrame** - 流畅的 60fps 动画

### 视觉效果
- 3D 透视轨道系统
- 音符旋转下落动画
- 打击波纹扩散特效
- 判定文字弹出动画
- 轨道发光脉冲效果
- 连击数字爆发动画

### 性能优化
- 对象池复用音符元素
- requestAnimationFrame 同步渲染
- CSS GPU 加速动画
- 按事件防抖处理

## 💡 游戏技巧

1. **保持节奏** - 跟随 BPM 的节拍感按键
2. **视觉预判** - 注意音符与判定线的距离
3. **连击积累** - 长连击能大幅提升分数
4. **放松手指** - 不要紧张按键，保持流畅
5. **选择合适难度** - 从 Easy 开始练习
