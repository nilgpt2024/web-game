# Sokoban (推箱子)

经典推箱子益智游戏，使用 CSS 3D transforms 实现立体视觉效果。

## 游戏玩法

将所有 **黄色箱子** 推到 **红色目标点** 上即可过关。

### 操作方式

| 按键 | 功能 |
|------|------|
| ↑ / W | 向上移动 |
| ↓ / S | 向下移动 |
| ← / A | 向左移动 |
| → / D | 向右移动 |
| Ctrl+Z | 撤销上一步 |
| R | 重新开始当前关卡 |
| Esc | 关闭规则面板 |

移动端支持屏幕方向按钮控制。

### 游戏规则

- 只能**推**箱子，不能拉
- 不能同时推两个或更多箱子
- 不能将箱子推入墙壁或另一个箱子
- 所有箱子到达目标点即获胜

## 游戏特色

- **12 个精心设计的关卡**，难度循序渐进
- **3D 立体视觉效果**：墙壁、箱子、玩家角色均采用 CSS 3D transforms 渲染
- **撤销功能**：支持无限步撤销，方便尝试不同策略
- **最优记录**：每关记录最佳步数，存储在本地
- **关卡选择**：自由选择已解锁的关卡
- **响应式设计**：适配桌面和移动设备
- **中英双语国际化**：支持中英文切换

## 技术实现

- 纯 HTML/CSS/JavaScript 实现，无外部依赖
- CSS 3D transforms（`perspective`、`rotateX/Y/Z`、`translateZ`）构建 3D 元素
- localStorage 存储游戏进度和最佳记录
- 关卡数据采用文本地图格式（# 墙壁, @ 玩家, $ 箱子, . 目标点）
