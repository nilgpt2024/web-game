# 🧩 Tangram 七巧板拼图游戏

一个基于 Web 的交互式七巧板（Tangram）拼图游戏，使用纯 HTML/CSS/JavaScript 实现，支持 3D 视觉效果、拖拽旋转操作和多个预设关卡。

## 游戏特色

- **7 块标准七巧板**：2 个大三角形、1 个中三角形、2 个小三角形、1 个正方形、1 个平行四边形
- **12 个预设图案**：房子、猫、帆船、跑步的人、天鹅、熊、椅子、箭头、鱼、跳舞的人、风筝、钻石
- **3D 视觉效果**：CSS 3D transforms + 投影阴影 + 渐变色彩
- **丰富的交互方式**：
  - 鼠标/触摸拖拽移动拼图块
  - 点击旋转 45°
  - 滚轮快速旋转
  - 平行四边形支持翻转（镜像）
- **智能吸附**：靠近正确位置时自动对齐
- **提示系统**：显示/隐藏目标轮廓
- **自由模式**：无限制自由创作
- **计时器 & 步数统计**
- **国际化支持**：中文 / English 双语切换
- **响应式设计**：适配桌面端和移动端

## 技术实现

- **CSS 3D Transforms**：`perspective` + `transform-style: preserve-3d` 实现立体感
- **SVG 目标图案**：使用 SVG `polygon` 绘制目标轮廓
- **事件处理**：统一 Mouse + Touch 事件支持多端操作
- **吸附算法**：欧氏距离检测 + 角度差判断自动对齐
- **胜利检测**：所有拼图块位置和角度在容差范围内即判定完成

## 浏览器兼容性

支持所有现代浏览器（Chrome、Firefox、Safari、Edge），需启用 JavaScript。
